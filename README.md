CAPTION AI : AI POWERED SOCIAL MEDIA CAPTION GENERATOR

**CaptionAI** is an AI-powered platform built with **Flask** and deployed via **Ngrok** that streamlines social media content creation. Driven by **Groq’s API** using the **LLaMA 3.3-70B Versatile** model, the system instantly processes user descriptions to deliver three unique captions, ten curated broad and niche hashtags, and three punchy call-to-action phrases tailored to the chosen platform and tone. Featuring secure API key management via `.env`, the application empowers marketers and creators to efficiently craft high-converting content with complete confidence.
